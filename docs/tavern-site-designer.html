Private communities should know where their data and rules live.

Tavern is self-hosted and intentionally avoids public discovery, federation, paid AI services, and external moderation vendors. Operators control infrastructure, policy, uploads, and access.

Local first Runs on infrastructure you control, with local fallbacks for Redis, object storage, ClamAV, and LiveKit when optional services are absent.
Safety core Access control, report queues, quarantine, audit logs, upload validation, and configurable policies are built into the product.
Honest limits No AI moderation claims, no public server discovery, no federation, no live transcription, and no monetization layer.

Operator visibility

Moderation queues, mass actions, audit trails, report dialogs, role editing, channel overwrites, and hidden-channel behavior are documented and wired.

Permission-aware

Gateway dispatch filtering respects channel access, hidden resources avoid existence leaks, and role overrides resolve deny before allow.

Upload hygiene

Magic-byte validation, image normalization, thumbnail generation, quarantine status, and optional ClamAV scanning keep file handling explicit.
